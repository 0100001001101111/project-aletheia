# Data Custodian: Pre-Flight Checklist

- [ ] Identity Selection: Public attribution or Anonymous?
- [ ] Data Audit: Does your dataset include Baseline, Stressor, and Outcome?
- [ ] Null Integrity: Have you included your null/failed trials?
- [ ] De-identification: All human subject identifiers removed?
- [ ] Schema Mapping: Which of the 5 domains matches your data?
- [ ] Format Check: Is data in JSON or ready for the LLM Parser?
- [ ] Self-Triage: Could a skeptic attempt replication with this data?
